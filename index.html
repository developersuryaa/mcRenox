HOW TO JOIN

Here you can see, how you can join on the Minecraft server.

1

Add the server

Go to Multiplayer in Minecraft, then Add Server

2

Enter server address

Enter mc.renoxdev.fun in the server address field and click "Done".

3

You're done!

Now you can play on mcRenox with other players!
